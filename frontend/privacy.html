Privacy & Security

Your data protection is our top priority

⚠️ Critical Disclaimer

TruthLens AI provides AI-based predictions and behavioral analysis only. This tool:

• Cannot claim 100% accuracy
• Is NOT a replacement for professional counseling or legal advice
• Should not be used as definitive proof in disputes
• Is for personal information purposes only
• Makes pattern-based predictions, not absolute truth claims

🔒 Data Protection

Zero-Storage Policy

We do not permanently store any conversation data. All processing happens in temporary session memory and is completely wiped after analysis completes.

Encryption

Data in transit is protected by TLS 1.3 encryption. Sensitive information (phone numbers, emails) is automatically stripped before processing.

No Third-Party Sharing

Your conversations are never shared with advertisers, data brokers, or third parties. Analysis is performed solely for your benefit.

✅ Security Measures

  • HTTPS/TLS encryption for all connections
  • Rate limiting to prevent abuse
  • Regular security audits and penetration testing
  • DDoS protection and WAF (Web Application Firewall)
  • Secure API authentication with OAuth 2.0
  • Regular dependency updates and patch management
  • Session timeout (auto-logout after 30 mins)

📋 GDPR Compliance

We comply with GDPR requirements:

  • • Lawful basis for processing (explicit user consent)
  • • Transparent data handling practices
  • • Right to data access (upon request)
  • • Right to be forgotten (data deletion)
  • • Minimal data collection
  • • Data protection impact assessments

🛡️ User Consent

By using TruthLens AI, you confirm:

  • ✓ You have explicit permission to analyze submitted conversations
  • ✓ You understand this is analysis, not definitive proof
  • ✓ You accept full responsibility for how you use these insights
  • ✓ You will not use this tool for harassment or illegal purposes

📞 Privacy Questions?

For privacy concerns or data requests, contact our privacy team at privacy@truthlens.ai
