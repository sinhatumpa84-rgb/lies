How TruthLens AI Works

A deep dive into our AI architecture and analysis methodology

AI Architecture

Input Processing

User uploads conversation text with explicit consent. System anonymizes and preprocesses data.

NLP Analysis

Advanced Natural Language Processing extracts linguistic features and sentiment patterns.

Pattern Detection

ML algorithms identify deception markers, emotional shifts, and behavioral anomalies.

Scoring Engine

Proprietary algorithm generates trustworthiness scores and risk metrics.

Report Generation

Creates comprehensive analysis with visualizations and actionable insights.

Privacy Cleanup

Data is immediately purged. No permanent storage or external sharing.

Detailed Workflow

Data Ingestion

User submits conversation (WhatsApp, Instagram, manually typed). System:

  • ✓ Validates input format
  • ✓ Removes PII (phone numbers, addresses)
  • ✓ Normalizes text encoding

Preprocessing

Text is cleaned and tokenized for analysis:

  • ✓ Lowercase conversion
  • ✓ Tokenization (words, sentences)
  • ✓ Stopword removal
  • ✓ Lemmatization

Feature Extraction

Extract linguistic and behavioral signals:

  • ✓ Sentiment scores (VADER/TransformerBERT)
  • ✓ Word frequency analysis
  • ✓ Pronoun usage patterns
  • ✓ Modifier intensity (very, definitely, etc.)
  • ✓ Question frequency
  • ✓ Message latency/gaps

Pattern Detection

ML model identifies behavioral patterns:

  • ✓ Logical contradictions
  • ✓ Evasion tactics
  • ✓ Emotional consistency
  • ✓ Response symmetry
  • ✓ Baseline deviation

Scoring & Analysis

Aggregate features into actionable metrics:

  • ✓ Trust Score (0-100)
  • ✓ Risk Indicators (High/Medium/Low)
  • ✓ Emotional Arc
  • ✓ Consistency Index
  • ✓ Engagement Level

Secure Cleanup

Immediate data purge after analysis:

  • ✓ Session data deleted
  • ✓ Temporary files wiped
  • ✓ Cache cleared
  • ✓ No backups retained

Technology Stack

NLP Libraries

  • • spaCy (tokenization, POS)
  • • NLTK (sentiment, analysis)
  • • Transformers (BERT/DistilBERT)
  • • VADER (sentiment intensity)

ML/AI Models

  • • OpenAI GPT-4 (insights)
  • • Scikit-learn (classification)
  • • LightGBM (pattern detection)
  • • XGBoost (scoring)

Backend

  • • FastAPI (Python)
  • • Node.js/Express (optional)
  • • Docker (containerization)
  • • Redis (caching)
